 11  Basic Genetic Algorithm (GA) 
 12   
 13  This sub-package implements a traditional genetic algorithm as described in 
 14  books and papers. It consists of selecting, breeding and mutating a population 
 15  of chromosomes (arrays of bits) and reinserting the fittest individual from the 
 16  previous generation if the GA is elitist. Please, consult a good reference on 
 17  the subject, for the subject is way too complicated to be explained here. 
 18   
 19  Within the algorithm implemented here, it is possible to specify and configure 
 20  the selection, crossover and mutation methods using the classes in the 
 21  respective sub-modules and custom methods can be implemented (check 
 22  ``selection``, ``crossover`` and ``mutation`` modules). 
 23   
 24  A GA object is actually a list of chromosomes. Please, refer to the 
 25  documentation of the class below for more information. 
 26  """

 46      A standard Genetic Algorithm 
 47   
 48      This class implements the methods to generate, initialize and evolve a 
 49      population of chromosomes according to a given fitness function. A standard 
 50      GA implements, in this order: 
 51   
 52        - A selection method, to choose, from this generation, which individuals 
 53          will be present in the next generation; 
 54        - A crossover method, to exchange information between selected individuals 
 55          to add diversity to the population; 
 56        - A mutation method, to change information in a selected individual, also 
 57          to add diversity to the population; 
 58        - The reinsertion of the fittest individual, if the population is elitist 
 59          (which is almost always the case). 
 60   
 61      A population is actually a list of chromosomes, and individuals can be 
 62      read and set as in a normal list. Use the ``[ ]`` operators to access 
 63      individual chromosomes but please be aware that modifying the information on 
 64      the list before the end of convergence can cause unpredictable results. The 
 65      population and the algorithm have also other properties, check below to see 
 66      more information on them.    '''
