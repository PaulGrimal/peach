 10  __doc__ = """ 
 11  Basic definitions and base classes for definition of fitness functions for use 
 12  with genetic algorithms. 
 13   
 14  Fitness is a function that rates higher the chromosomes that perform better 
 15  according to the objective function. For example, if the minimum of a function 
 16  needs to be found, then the fitness function should rate better the chromosomes 
 17  that correspond to lower values of the objective function. This module gives 
 18  support to use common Python functions as fitness functions in genetic 
 19  algorithms. 
 20   
 21  The classes defined in this sub-module take a function and use some algorithm to 
 22  rank a population. There are some different ranking functions, some are provided 
 23  in this module. There is also a class that can be subclassed to generate other 
 24  fitness methods. See the documentation of the corresponding class for more 
 25  information. 
 26  """
